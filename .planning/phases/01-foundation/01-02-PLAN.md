**Class: VirtualPortfolio**

```js
'use strict';

class VirtualPortfolio {
  constructor(initialCapital = 1000000) {
    this.initialCapital = initialCapital;
    this.availableCapital = initialCapital;
    // positions: Map<symbol, { qty: number, totalCost: number }>
    // totalCost = sum of all fill prices * qty (for weighted avg cost)
    this._positions = new Map();
    // trades: array of trade records
    this._trades = [];
    this._tradeCounter = 0;
  }
```

**`buy(symbol, qty, price, slippage = 0.001)`:**
- Validate: `qty > 0` else throw `Error('qty must be positive')`
- Compute fill price: `const fillPrice = parseFloat((price * (1 + slippage)).toFixed(2))`
- Compute total cost: `const cost = fillPrice * qty`
- Guard capital: if `cost > this.availableCapital` throw `Error('Insufficient capital: need ' + cost.toFixed(2) + ', have ' + this.availableCapital.toFixed(2))`
- Deduct capital: `this.availableCapital -= cost`
- Update position: if symbol not in `_positions`, set `{ qty: 0, totalCost: 0 }`. Then: `pos.qty += qty; pos.totalCost += cost`
- Record trade: push to `_trades`:
```js
{
  id: ++this._tradeCounter,
  symbol,
  side: 'BUY',
  qty,
  fillPrice,
  slippage,
  cost,
  timestamp: Date.now(),
}
```
- Return the trade record

**`sell(symbol, qty, price, slippage = 0.001)`:**
- Validate: `qty > 0` else throw `Error('qty must be positive')`
- Get position: `const pos = this._positions.get(symbol)`
- Guard position existence and qty: if `!pos || pos.qty < qty` throw `Error('Insufficient position: held ' + (pos?.qty || 0) + ', attempting to sell ' + qty)`
- Compute fill price: `const fillPrice = parseFloat((price * (1 - slippage)).toFixed(2))`
- Compute avg cost per unit: `const avgCost = pos.totalCost / pos.qty`
- Compute realized P&L for this sale: `const realizedPnl = (fillPrice - avgCost) * qty`
- Compute proceeds: `const proceeds = fillPrice * qty`
- Return capital: `this.availableCapital += proceeds`
- Update position: `pos.qty -= qty; pos.totalCost -= avgCost * qty`
- If `pos.qty === 0`, delete from `_positions`
- Record trade: push to `_trades`:
```js
{
  id: ++this._tradeCounter,
  symbol,
  side: 'SELL',
  qty,
  fillPrice,
  slippage,
  proceeds,
  realizedPnl,
  timestamp: Date.now(),
}
```
- Return the trade record

**`getPositions()`:**
- Return array from `_positions` entries:
```js
return Array.from(this._positions.entries()).map(([symbol, pos]) => ({
  symbol,
  qty: pos.qty,
  avgCost: parseFloat((pos.totalCost / pos.qty).toFixed(2)),
  totalCost: parseFloat(pos.totalCost.toFixed(2)),
}));
```

**`getPnL(currentPrices = {})`:**
- `currentPrices` shape: `{ 'NIFTY 50': 22550.00, 'BANK NIFTY': 47300.00 }`
- Compute unrealized:
```js
let unrealized = 0;
for (const [symbol, pos] of this._positions.entries()) {
  const ltp = currentPrices[symbol];
  if (ltp == null) continue; // skip if no live price
  const avgCost = pos.totalCost / pos.qty;
  unrealized += (ltp - avgCost) * pos.qty;
}
unrealized = parseFloat(unrealized.toFixed(2));
```
- Compute realized: sum all `trade.realizedPnl` values from `_trades` where `trade.side === 'SELL'`:
```js
const realized = parseFloat(
  this._trades
    .filter(t => t.side === 'SELL')
    .reduce((sum, t) => sum + (t.realizedPnl || 0), 0)
    .toFixed(2)
);
```
- Return:
```js
return {
  realized,
  unrealized,
  total: parseFloat((realized + unrealized).toFixed(2)),
  capital: this.initialCapital,
  availableCapital: parseFloat(this.availableCapital.toFixed(2)),
};
```

**`getTrades()`:**
- Return `[...this._trades]` (shallow copy, chronological order)

**`getSummary(currentPrices = {})`:**
- Convenience method returning:
```js
return {
  ...this.getPnL(currentPrices),
  positions: this.getPositions(),
  tradeCount: this._trades.length,
};
```

**Export:**

module.exports = { VirtualPortfolio };
