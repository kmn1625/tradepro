**Class: KotakFeedService**

Constructor:
```js
constructor() {
  this._ws = null;
  this._broadcastFn = null;
  this._reconnectTimer = null;
  this._connected = false;
  this._pingInterval = null;
}
```

Public method `startFeed(broadcastFn)`:
- Stores `broadcastFn` as `this._broadcastFn`
- Calls `this._connect()`

Private method `_connect()`:
- Read `session.getSession()`. If `!session.isAuthenticated()`, log a warning: `"[KotakFeed] Session not authenticated — feed not started. Call POST /api/auth/login first."` and return without connecting. Do NOT throw.
- Retrieve `accessToken` from session.
- Construct WebSocket URL: `wss://mlhsm.kotaksecurities.com`
- Connect with headers: `{ Authorization: 'Bearer ' + accessToken }` passed as the third argument to `new WebSocket(url, [], { headers: { Authorization: ... } })`
- On `open`: log `"[KotakFeed] Connected"`, set `this._connected = true`, call `this._subscribe()`, start ping every 25s with `this._pingInterval = setInterval(() => this._ws.ping(), 25000)`
- On `message`: call `this._handleTick(data)`
- On `close`: log `"[KotakFeed] Disconnected — reconnecting in 5s"`, set `this._connected = false`, clear ping interval, schedule `this._reconnectTimer = setTimeout(() => this._connect(), 5000)`
- On `error`: log `"[KotakFeed] WS error: " + err.message`, do NOT crash — close event follows

Private method `_subscribe()`:
- Build subscription message per Kotak Neo streaming protocol:
```js
const tokens = Object.values(INSTRUMENT_TOKENS);
const msg = JSON.stringify({
  type: 'subscribe',
  tokens: tokens,   // ['26000', '26009', '26037']
});
this._ws.send(msg);
```

Private method `_handleTick(rawData)`:
- Parse JSON: `const tick = JSON.parse(rawData.toString())`
- Guard: if `!tick || !tick.token` return immediately (skip ping-pong frames)
- Resolve symbol: `const symbol = Object.entries(INSTRUMENT_TOKENS).find(([, t]) => t === String(tick.token))?.[0]`
- If no symbol found, return
- Extract price: `const price = parseFloat(tick.ltp || tick.last_price || tick.ltP || 0)`
- If price <= 0, return
- Call `marketDataService.processTick(symbol, price, Date.now())`
- Broadcast raw tick:
```js
this._broadcastFn({ type: 'PRICE_UPDATE', symbol, price, time: Date.now() });
```
- Broadcast partial candle for '1m' and '5m':
```js
for (const interval of ['1m', '5m']) {
  const partial = marketDataService.partialCandle[symbol]?.[interval];
  if (partial) {
    this._broadcastFn({
      type: 'CANDLE_UPDATE', symbol, interval,
      time: partial.time,
      open: parseFloat(partial.open.toFixed(2)),
      high: parseFloat(partial.high.toFixed(2)),
      low: parseFloat(partial.low.toFixed(2)),
      close: parseFloat(partial.close.toFixed(2)),
    });
  }
}
```

Public method `stopFeed()`:
- Clear `this._pingInterval` and `this._reconnectTimer`
- If `this._ws`, call `this._ws.terminate()`
- Set `this._connected = false`

Export singleton:
```js
module.exports = new KotakFeedService();
```

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Kotak WS server → backend | Untrusted wire data; tick payloads must be validated before use |
| Backend memory → broadcast | Price values flow to all WebSocket clients; NaN/Infinity must not propagate |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-01 | Tampering | `_handleTick` price parsing | mitigate | Validate `price > 0` before calling processTick; parseFloat guard already in action |
| T-01-02 | Denial of Service | WebSocket reconnect loop | mitigate | 5s back-off timer prevents rapid reconnect storm; stopFeed() clears timer |
| T-01-03 | Information Disclosure | accessToken in WS header | accept | Token sent only to Kotak's own endpoint over TLS (wss://); low risk |
| T-01-04 | Denial of Service | Malformed JSON from Kotak | mitigate | `JSON.parse` inside try/catch implied by `if (!tick || !tick.token) return` guard pattern |
</threat_model>
